# Aplicativo de Planejamento de Viagens - Simples

## Funcionalidades
- Cadastrar **destinos** (nome, país, custo médio diário)
- Criar **itinerários** (título, destino, data de início e fim)

## Como rodar
1. Instalar Node.js (LTS)
2. Abrir pasta no VS Code
3. Rodar `npm install`
4. Rodar `npm start`
